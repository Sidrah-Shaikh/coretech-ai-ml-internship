# Task 7 - Natural Language Processing Basics

## Objective
Perform basic NLP tasks and sentiment analysis using Naive Bayes.

## Dataset
Created custom feedback dataset with 30 records.

Columns:
- Feedback_ID
- Client_Name
- Service
- Feedback_Text
- Rating
- Sentiment

## NLP Steps
1. Text Cleaning
2. Lowercasing
3. Remove Punctuation
4. Remove Stopwords
5. Tokenization
6. Stemming
7. TF-IDF Vectorization

## Machine Learning
- Naive Bayes Classifier

## Visualizations
- Sentiment Distribution Chart
- Word Cloud
- Confusion Matrix
