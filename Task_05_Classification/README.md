# Task 5 - Supervised Learning Classification

## Objective
Build and evaluate machine learning classification models to predict client project status.

## Dataset
- 50 records
- Features:
  - Budget
  - Team_Size
  - Duration
- Target:
  - Status

## Models Used
1. Logistic Regression
2. Decision Tree
3. Random Forest

## Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report

## Output Files
- dataset.csv
- model_comparison.csv
- classification_report.txt
- confusion_matrix.png

## Best Model
Random Forest
