# Task 6: Unsupervised Learning - Clustering

## Objective

The objective of this project is to perform customer segmentation using clustering techniques. The dataset contains customer information including Age, Annual Income, and Spending Score.

## Dataset

The dataset contains 50 customer records with the following features:

* ClientID
* Age
* AnnualIncome
* SpendingScore

## Data Preprocessing

The following preprocessing steps were performed:

* Loaded the dataset using Pandas.
* Checked dataset information.
* Checked for missing values.
* Generated descriptive statistics.
* Selected numerical features.
* Applied StandardScaler for feature scaling.

## Elbow Method

The Elbow Method was used to determine the optimal number of clusters. The graph showed an elbow point at K = 3, indicating that three clusters were appropriate for the dataset.

## K-Means Clustering

K-Means clustering was applied with 3 clusters. Each customer was assigned to a cluster based on similarity in age, income, and spending behavior.

## Cluster Visualization

A scatter plot was created to visualize the clusters generated by the K-Means algorithm.

## Cluster Analysis

### Cluster 0

High income and high spending customers.

### Cluster 1

Medium income and medium spending customers.

### Cluster 2

Low income and low spending customers.

## Silhouette Score

The Silhouette Score was calculated to evaluate cluster quality. A higher score indicates better separation between clusters.

## Hierarchical Clustering

Agglomerative Hierarchical Clustering was also applied with 3 clusters and the results were compared with K-Means clustering.

## Comparison of Methods

| Method                  | Observation                                                |
| ----------------------- | ---------------------------------------------------------- |
| K-Means                 | Fast and efficient                                         |
| Hierarchical Clustering | Useful for understanding relationships between data points |

## Business Insights

* Premium customers can be targeted with VIP offers.
* Regular customers can receive loyalty rewards.
* Low-spending customers can be encouraged through promotional discounts.

## Conclusion

Customer segmentation was successfully performed using K-Means and Hierarchical Clustering techniques. Three distinct customer groups were identified, which can help businesses design targeted marketing strategies and improve customer engagement.
